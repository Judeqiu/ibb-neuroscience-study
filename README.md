# IB Neuroscience Study Hub

Interactive study page for IB Neuroscience — deep notes, glossary, pop quiz, blog, and source PDFs.

## Features

- **Deep Notes** with glossary cross-referencing (hover for definitions, click to jump)
- **Glossary** with real-time search and term highlighting
- **Pop Quiz** with reveal-answer interactivity (13 questions)
- **Source PDFs** — 4 original textbooks embedded
- **Collapsible sidebar** with localStorage persistence
- **Responsive** — works on desktop and mobile

## Open

Open `index.html` in a browser, or visit the GitHub Pages URL.

## Source

Content generated from IBB Neuroscience curriculum materials (Neurohistology, Neuroanatomy, Neurology) by Charles Watson et al.
